## About Hyamii

Hyamii (formerly TableTrack) is a comprehensive SaaS-based restaurant management solution that revolutionizes how restaurants operate. Built with **Laravel** and **Livewire**, it offers a modern, responsive interface for seamless restaurant management.

Whether you're running a single restaurant or managing multiple locations, Hyamii provides all the tools you need to streamline operations, improve efficiency, and grow your business.

## Key Features

- **Multi-Restaurant Management** - Manage multiple restaurant locations from a single dashboard
- **Table & Reservation Management** - Efficiently manage tables, reservations, and seating
- **Menu & Order Management** - Complete menu management with real-time order processing
- **Kitchen Display System (KDS)** - Streamline kitchen operations with digital order displays
- **Inventory Management** - Track stock levels, manage suppliers, and automate reordering
- **Staff Management & Scheduling** - Manage staff, roles, permissions, and schedules
- **Real-time Analytics & Reports** - Comprehensive reporting and analytics dashboard
- **Multi-language Support** - Support for multiple languages and currencies
- **Subscription & Payment Integration** - Built-in subscription management and payment gateways
- **Cash Register Module** - Complete cash register and transaction management
- **Kiosk Mode** - Self-service kiosk for customers
- **SMS Notifications** - Automated SMS notifications for orders and updates
- **Multi-POS Support** - Multiple point-of-sale locations management

## Technology Stack

- **Framework:** Laravel
- **Frontend:** Livewire, Vue 3, Tailwind CSS
- **Database:** MySQL/PostgreSQL
- **Payment Gateways:** Stripe, PayPal, Mollie, Flutterwave, Paystack, Razorpay, Xendit, Payfast, Tap

## Requirements

- PHP >= 8.2
- MySQL >= 5.7 
- Composer
- Node.js & NPM
- Web Server (Apache/Nginx)

## Modules

Hyamii comes with several powerful modules:
